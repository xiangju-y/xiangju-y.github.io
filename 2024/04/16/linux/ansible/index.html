2.4 file模块

  • 管理文件,管理目录,软连接
  • 详细看这个https://docs.ansible.com/ansible/2.9/modules/file_module.html

参数：ansible all -m file -a ‘path=/opt/li.txt state=touch’

选项
path 路径 文件或者目录的路径
state状态 ➖touch创建文件
➖directory创建目录
➖absent删除目标文件
➖file(默认)更新文件
➖hard创建或更改硬链接与src一起使用
➖link创建或更改软链接与src一起使用
src 要链接到的文件的路径
owner 设备目标文件的所属主
mode 生成的文件或目录应具有的权限
grounp 设备目标文件的所属组
